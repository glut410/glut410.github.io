词法分析

主要任务

  • 扫描单词，确定单词类型。
  • 将单词转化为tokens。

计算机中的单词类型

  • 关键字 Keyword

    例如：if、else、for、u8（如果你不知道什么是u8，这明显地触怒了Saint He）…

    关于什么是u8，这我得好好说道说道。首先呢，“u”和“v”很像，所以是v8，请关注嘉然今天吃什么，关注嘉然，顿顿解馋~关注永雏塔菲喵，关注永雏塔菲谢谢喵

  • 标识符 Identifier

    例如：变量名、方法名…

  • 常量 Const

    例如：114514.19180，true

  • 运算符 Operator

    例如：+、-、*、/、++、>、<、|（在著名的Helang编程语言中，这不仅仅只是或运算。这个运算符的发明，大大提高了开发效率，是划时代的里程碑）

  • 界限符 Delimiter

    例如：; ( ) = { } …
